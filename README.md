# Macro-Recorder
You can record, save, load all the in/output of your mouse and keyboard (no mouse scrolling sorry)

How To Use:
Go to Powershell and 'pip install pyautogui'
Then, run the code.
Press 'Record' and then you can do anything you want, when done, press 'Stop'.
If you want insta-replay, press 'Replay' afterwards.If you want to save, then press 'Save' to a file, next time when you want to use it, press 'Load' then the file (.json) and press 'Replay'.
The 'Loop' is for infinite replaying automatation.Use F8 or (emergancy) Alt+F4 to force stop.
